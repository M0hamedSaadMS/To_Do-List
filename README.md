# 📋 ToDo List Application

A lightweight desktop task manager built with Java Swing and file-based persistence.

---

## 📖 Overview

This application allows users to create, track, and persist tasks through a graphical interface. Tasks are automatically saved to a local file and restored on application startup.

**⏱️ Time to build:** ~2 hours  
**🎯 Purpose:** Personal practice with Java GUI development and file I/O

## ✨ Features

| Feature | Description |
|---------|-------------|
| ➕ Create | Add new tasks to your list |
| 🗑️ Delete | Remove unwanted tasks |
| ✅ Complete | Mark tasks as done (single-click or double-click) |
| 💾 Auto-save | Automatically saves to `tasks.txt` |
| 🔄 Persistent | Tasks reload on application startup |
| 🎨 Dark theme | Built-in dark mode interface |
